Beispiele SpringSource dm Server

Installation der SpringSource dm Server Beispiel

Die Beispiele für den dm Server stehen hier zum Download zur Verfügung. Das Petclininc-Beispiel finden Sie beim SpringSource dm Server Download. Als Entwicklungsumgebung sollten Sie die SpringSource Tool Suite nutzen, die kostenlos verfügbar ist.

Nun sollten Sie den SpringSource dm Server installieren:

  • Laden Sie den SpringSource dm Server 1.0 von http://springsource.org/dmserver herunter und entpacken Sie ihn auf Ihre Festplatte.
  • Starten Sie die SpringSource ToolSuite mit einem leeren Workspace.
  • Wählen Sie Windows->Show View->Other...
  • In dem nun erscheinenden Dialog "Show View" wählen Sie aus dem Server-Bereich die View Servers.
  • In der nun erschienenden Servers-View klicken Sie mit der rechten Maustaste und wählen New->Server
  • Wählen Sie in dem nun erscheinden "New Server" dialog unter SpringSource den "SpringSource dm Server 1.0" aus und klicken Sie auf Next.
  • Wählen Sie nun den Pfad, unter dem Sie den dm Server entpackt haben, mit Browse... aus.
  • Wählen Sie Finish.

Sie müssen nun in den SpringSource dm Server noch einige Bibliotheken installieren:

  • Doppelklicken Sie auf den SpringSource dm Server v1.0 im Servers-View.
  • Wählen Sie in der neu erschienden View das Repository Tab.
  • Dort können Sie nun das Repository durchsuchen. Folgende Bibliotheken müssen Sie installieren:
    • com.springsource.org.apache.commons.dbcp (1.2.2.osgi)
    • com.springsource.org.apache.commons.pool (1.4.0)
    • com.springsource.org.eclipse.persistence (1.0.0)
    • com.springsource.org.eclipse.persistence.jpa (1.0.0)
    • com.springsource.org.eclipse.persistence.asm (1.0.0)
    • com.springsource.org.eclipse.persistence.antlr (1.0.0)
    • com.springsource.org.hsqldb (1.8.0.9)

Um die Anwendungen zu installieren, müssen sie folgendes tun:

  • Wählen Sie File->Import...
  • In dem nun erscheinenden Import-Dialog wählen Sie unter General "Existing Projects into Workspace"
  • In dem Dialog-Fenster wählen Sie "Select Archive File" und dann mit Browse... die heruntergeladenen ZIP-Datei.
  • Wählen Sie Finish

Die Projekte sind nun in Ihrem Workspace und sie können Sie auf dem dm Server installieren und in den Code schauen.

Beispielsweise können Sie nun das Projekt "de.spring_buch" auf den Server ziehen und dann den Server mit einem rechten Mausklick starten.

Anschließend steht die Anwendung unter http://localhost:8080/spring-buch zur Verfügung.
