Dritte Auflage

Die Beispiele für die dritte Auflage des Buches sind hier. Bitte lesen Sie das enthaltene LiesMich.html. Die Installation zeigt folgendes Video: http://www.youtube.com/watch?v=A8QacKO2944 .

Außerdem finden Sie die Beispiele bei github: https://github.com/ewolff/Spring-Buch . Dort kann man die Sourcen als Archiv herunterladen. Oder man nutzt direkt die Versionkontrolle git, um eine lokale Kopie anzulegen: git clone git://github.com/ewolff/Spring-Buch.git .

Wenn Sie ein neues Spring-Projekt aufbauen wollen, empfehle ich Ihnen auch einen Blick auf Spring Roo. Ein Video dazu findet sich unter http://www.youtube.com/watch?v=y1hbCR4KT44.
