Über den Autor

Eberhard Wolff

Eberhard Wolff beschäftigt sich mit Java seit der Version 1.0. Seine Arbeitsschwerpunkte sind Java, Java EE und Komponentenarchitekturen. Außerdem hält er regelmäßig Vorträge und schreibt Fachartikel im Java-Umfeld. Als Mitautor war er bereits beteiligt an den Büchern "Server Component Patterns" (2002) und "Java-Persistenz-Strategien" (2004).

Er ist Gründungsmitglied der Java Champions, einer Gruppe von Java-Experten, die Input zur Plattform liefern und die weitere Verbreitung von Java unterstützen. Er arbeitet für die adesso AG als Architecture and Technology Manager. In dieser Funktion arbeitet er an der technologischen Ausrichtung der Firma. Er ist Autor verschiedener Fachartikel und Sprecher auf internationalen Konferenzen.
